Detailed Description

This class holds a dynamic list of Tools which can be Initialised, Executed and Finalised to perform program operation. Large number of options in terms of run modes and setting can be assigned.

Constructor with explicit configuration variables passed as arguments.

Parameters
verboseThe verbosity level of the ToolChain. The higher the number the more explicit the print outs. 0 = silent apart from errors.
errorlevelThe behavior that occurs when the ToolChain encounters an error. 0 = do not exit for both handeled and unhandeled errors, 1 = exit on unhandeled errors only, 2 = exit on handeled and unhandeled errors.
serviceThe name of the ToolChain service to boradcast for service discovery
logmodeWhere log printouts should be forwarded too. "Interactive" = cout, "Remote" = send to a remote logging system, "local" = ouput logs to a file.
log_local_pathThe file path and name of where to store logs if in Local logging mode.
log_serviceService name of remote logging system if in Remote logging mode.
log_portThe Port number of the remote logging service
pub_secThe number of seconds between publishing multicast serivce beacons
kick_secThe number of seconds to wait before removing a servic from the remote services list if no beacon received.
IO_ThreadsThe number of ZMQ IO threads to use (~1 per Gbps of traffic).
