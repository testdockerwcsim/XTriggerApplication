24 ## Configuration
25 
26 ```
27 trigger_search_window WINDOW
28 trigger_threshold THRESHOLD
29 trigger_threshold_adjust_for_noise BOOL
30 pretrigger_save_window PRETRIGGER
31 posttrigger_save_window POSTTRIGGER
32 trigger_od BOOL
33 use_stopwatch BOOL
34 stopwatch_file FILENAME
35 verbose LEVEL
36 ```
37 * `trigger_search_window` Width of the sliding window, in ns
38 * `trigger_threshold` Trigger threshold - number of triggers must be above this value (equal to does not fire the trigger)
39 * `trigger_threshold_adjust_for_noise` Add the average dark noise rate to the threshold?
40 * `pretrigger_save_window` After a trigger is found, save digits from `trigger_time - pretrigger_save_window` to `trigger_time + posttrigger_save_window`
41 * `posttrigger_save_window` After a trigger is found, save digits from `trigger_time - pretrigger_save_window` to `trigger_time + posttrigger_save_window`
42 * `trigger_od` Trigger on OD digits, rather than ID digits?
43 * `use_stopwatch` Use the Stopwatch functionality implemented for this tool?
44 * `stopwatch_file` Save the time it takes for each run of `Execute()` to a histogram. Should end in .pdf, .eps, etc.
45 * `verbose` Verbosity level. Runs from 0 (low verbosity) to 9 (high verbosity)
